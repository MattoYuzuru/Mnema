# Политика безопасности

## Поддерживаемые версии

Актуальные версии проекта, для которых выпускаются обновления безопасности:

| Версия  | Поддерживается     |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Сообщение об уязвимостях

Если вы нашли уязвимость в проекте, пожалуйста, сообщите об этом через issues или напрямую разработчику.
Мы постараемся ответить в течение нескольких дней и сообщить о ходе исправления.

Проект является учебным (курсовая работа) и представляет собой веб-приложение для эффективного запоминания информации с помощью интерактивных карточек, поэтому обновления безопасности публикуются по мере возможности.
